1. Операції зі строками.

Строка - це набір символів, обмежених одинарними чи подвійними лапками.

Створення строк

var a = 'hello';
var b = "hello";
var c = 13 + '';
var d = 42.5;
d = d.toString();
console.log(a, b, c, d);

Метод .toString() - найкоректніший спосіб перетворити будь-яку змінну в строку.

Не плутайте назви змінних і строки:

var hello = 42;
console.log(hello);
console.log('hello');

Дії зі строками.

Строки можна об'єднувати, це називається конкатинацією строк.

var a = "Hello," + ' world!';
var n1 = 'Іванов', n2 = 'Іван', n3 = 'Іванович';
var n = n1 + ' ' + n2 + ' ' + n3;
console.log(a);
console.log(n);

Строку можна об'єднати з числом, змінною, об'єктом. Це все перетвориться в строку.

var b = 64 + ' кг.';
var price = 42, c = 'Ціна: ' + price + ' грн.';
var d = 'Масив: ' + [1, 2, 3];
var und, e = "Не задано: " + und;
console.log(b);
console.log(c);
console.log(d);
console.log(e);

Спецсимволи.

Зворотній слеш слугує для створення спеціальних символів. Це називається екрануванням символів.

console.log('Hello,\nworld');
console.log('Кам\'янець Подільський');
console.log("Газета \"Проскурів\"");
console.log('D:\\work\\frontend\\');

За допомогою зворотнього слеша можна представити будь-який символ юнікода, вказавши його номер в таблиці символів в 16-й системі зчислення (hex):

console.log('\u00A9 \u03C0 \u2764');

Доступ до символів.

Звернутися до кожного символа можна через його індекс (номер, починаючи з нуля) в квадратних дужках після змінної:

var s = 'Hello!';
console.log(s[0]);
console.log(s[1]);
console.log(s[2]);
console.log(s[3]);

У строки є властивість length, в ній зберігається число - кількість символів в строці, довжина строки.

console.log(s.length);
var n = s.length;
console.log(s[n-1]);    // останній символ строки
console.log(s[n-2]);    // передостанній символ строки

console.log(s[s.length-1]);

В JavaScript літери в строках не можна змінювати. Наступний код не призведе до зміни строки:

var s = 'вода';
s[0] = 'с';
console.log(s);

2. Знайомство з циклом for.

Якщо нам треба перебрати декілька елементів - можна скористатися циклом.

Цикли повторюють набір команд певну кількість разів, задану через умову.

Є декілька видів циклів, один з них - for:

for ( ініціалізація лічильника ; умова ; інкремент лічильника ) {
  тіло цикла;
}

for (var i = 1; i < 5; i = i + 1) {
  console.log(i);
}
var s = 'Маша';
for (i = 3; i >= 0; i--) {
  console.log(s[i]);
}

Кожне виконання команд, кожен прохід по циклу називається ітерацією циклу.

Лічильник також називають ітератором.

Увага! Будьте обережні! Якщо невірно задати умову - може створитися безкінечний цикл, при цьому браузер підвисає і його доведеться перезапустити.

    а 1
    ш 2
    а 3
  2. Створіть пусту строку і циклом занесіть у неї цифри від 0 до 5. Виведіть строку в alert, повинно відобразитися "012345".
  3. Виведіть в консоль 10 випадкових чисел від 0 до 1. Використайте групування в консолі.
  4. Цикли повинні бути універсальними: якщо зміниться строка (довжина строки), то скрипт повинен відпрацювати без помилок.

3. Методи строк.

Методи застосовуються до змінних, або до самих строк. Прописуються через крапку:

var s = 'Іванов Іван Іванович';
var m = s.split();
'Hello!'.toUpperCase();

Дослідіть результат роботи наступного коду:

console.dir(Object(''));
console.dir(Object('Hello!'));

.toLowerCase(), .toUpperCase() - повертають строку, в якій змінено регістр.

var s = "Привіт, Світ!";
console.log(s.toLowerCase());
var T = s.toUpperCase();
console.log(T);

.substring(a, b), .substr(a, b), .slice(a, b) - повертають частину строки.

var s = 'маргарин';
console.log(s.substring(3));
console.log(s.substring(3, 6));
console.log(s.slice(3));
console.log(s.slice(3, 6));
console.log(s.substr(3));
console.log(s.substr(3, 3));

Можна задавати від'ємні значення - в такому випадку нумерація символів ведеться з кінця строки.

.indexOf(c, n), .lastIndexOf(c, n) - повертають індекс підстроки в строці.

Якщо підстроку не знайдено - буде повернено -1.

var s = 'маргарин';
console.log(s.indexOf('a'));
console.log(s.indexOf('гaр'));
console.log(s.indexOf('a', 2));
console.log(s.lastIndexOf('a'));

.split(sep) - розбиває строки по роздільнику (сепаратору), повертає масив.

var s = '5, 12, привіт, 42'; var m = s.split(', '); console.log(m);

 

.charCodeAt(n) - повертає код символа.

var s = 'abc';
console.log(s.charCodeAt(0));
console.log(s.charCodeAt(1));
console.log(s.charCodeAt(2));

Обернена функція:

var a = String.fromCharCode(97);

Застарілі або рідковживані методи в лекції не представлено.

Методи для роботи з регулярними виразами будуть розглянуті на наступній лекції.
