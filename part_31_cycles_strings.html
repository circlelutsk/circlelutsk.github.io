Цикли, методи строк.

1. Цикл for: вічний цикл, break, continue.

В циклі можна не створювати лічильник, якщо він був створений раніше:

var i;
for (var i = 0; i < 8; i++) { ... }
for (i = 0; i < 8; i++) { ... }
for (i = 5; i > 0; i--) { ... }

В циклі можна не задавати лічильник, якщо його задано раніше.

var i = s.length - 1;
for (; i >= 0; i++) { ... }

В циклі можна не задавати зміну лічильника, якщо лічильник буде мінятися в тілі циклу:

var i = 0;
for (; i < 10;) {
  ...
  i++;
}

В циклі можна не задавати умову, в результаті у нас вийде безкінечний цикл:

for (;;) { console.log('Браузер підвис'); }

Команда break дозволяє обірвати цикл.

Наприклад, наступний код шукає букву d в тексті. Знайшли букву - далі перебирати строку немає потреби:

var s = 'Lorem ipsum dolor sit amet';
for (var i = 0; i < s.length; i++) {
  if (s[i] === 'd') { break; }
}
console.log('Буква d на ' + i + ' місці.');    // починаючи з одиниці

Команда continue дозволяє закінчити поточну ітерацію циклу і перейти на наступну.

Наступний код буде виконувати якісь дії лише з простими числами:

for (var i = 1; i < 1000000; i++) {
  if (!ifSimple(i)) { continue; }
  // дії над простим числом:
  ...
}
function ifSimple(n) { повернути true якщо число n просте, false - якщо ні }

2. Цикл while.

Даний цикл має в собі тільки умову. Поки вона ствердна - виконуються ітерації циклу.

Керування ітератором здійснюється в тілі циклу, а його ініціалізація - перед циклом.

Ітератор не є обов'язковим, якщо не потрібно рахувати ітерації.

while ( умова ) {
  тіло циклу;
}

var i = 0;
while (i < s.length) {
  ...
  i++;
}

Вічний цикл while:

while (1) {
  ...
}

У всіх циклах можна застосовувати оператори break та continue.

3. Цикл do-while.

Даний цикл спочатку виконує ітерацію, а в кінці перевіряє чи виконувати наступну.

Попередні цикли можуть не виконати ні одної ітерації, якщо умова одразу не виконується.

Цикл do-while мінімум одну ітерацію виконає.

do {
  тіло циклу;
} while ( умова );

4. Методи строк.

Методи застосовуються до змінних, або до самих строк. Прописуються через крапку:

var s = 'Іванов Іван Іванович';
var m = s.split();
'Hello!'.toUpperCase();

Дослідіть результат роботи наступного коду:

console.dir(Object(''));
console.dir(Object('Hello!'));

.toLowerCase(), .toUpperCase() - повертають строку, в якій змінено регістр.

var s = "Привіт, Світ!";
console.log(s.toLowerCase());
var T = s.toUpperCase();
console.log(T);

.substring(a, b), .substr(a, b), .slice(a, b) - повертають частину строки.

var s = 'маргарин';
console.log(s.substring(3));
console.log(s.substring(3, 6));
console.log(s.slice(3));
console.log(s.slice(3, 6));
console.log(s.substr(3));
console.log(s.substr(3, 3));

Можна задавати від'ємні значення - в такому випадку нумерація символів ведеться з кінця строки.

.indexOf(c, n), .lastIndexOf(c, n) - повертають індекс підстроки в строці.

Якщо підстроку не знайдено - буде повернено -1.

var s = 'маргарин';
console.log(s.indexOf('a'));
console.log(s.indexOf('гaр'));
console.log(s.indexOf('a', 2));
console.log(s.lastIndexOf('a'));

.split(sep) - розбиває строки по роздільнику (сепаратору), повертає масив.

var s = '5, 12, привіт, 42';
var m = s.split(', ');
console.log(m);

 

.charCodeAt(n) - повертає код символа.

var s = 'abc';
console.log(s.charCodeAt(0));
console.log(s.charCodeAt(1));
console.log(s.charCodeAt(2));

Обернена функція:

var a = String.fromCharCode(97);

.trim() - обрізає пробіли з початку і з кінця строки.

<input type="email" name="email" id="email">

var email = document.getElementById('email').value.trim();

Застарілі або рідковживані методи в лекції не представлено.

Методи для роботи з регулярними виразами будуть розглянуті на наступній лекції.
