1. Включення JS у веб-сторінку.

Код JS може включатися безпосередньо у веб-сторінку в розділ <head> чи в тіло документа:

<html>
<head>
  <script type="text/javascript">
    var blahblah = 34;
  </script>
</head>
<body>
  <script type="text/javascript">
    document.write('<p>Hello!</p>');
  </script>
</body>
</html>

Також JS може підключатися з файла, знову ж таки в розділі <head> чи в тілі документа:

<html>
<head>
  <script type="text/javascript" src="js/my.js?update_05_01_2016_1"></script>
</head>
<body>
  <script type="text/javascript" src="js/jQuery.min.js"></script>
</body>
</html>

Часто скрипти підключають в кінці веб-сторінки для того, щоб вони починалися грузитися в самому кінці.

Реакції на події можна включати у вигляді інлайнових скриптів, проте, це погана практика:

<button onclick="alert('Ви клацнули по кнопці.');">Клацни сюди!</button>

2. Консоль розробника та вивід інформації для розробника.

Консоль - зручний інструмент для відлагодження скриптів, пошуку помилок, виводу службової інформації.

Консоль викликається клавішою F12 (якщо у вас замість появи консолі пропав інтернет - спробуйте комбінацію fn + F12).

alert('hello');
console.log('hello');
console.dir('hello');
console.info('hello');
console.warn('hello');
console.error('hello');

Різницю між log та dir можна побачити згодом, при роботі з об'єктами:

<p id="hey">Hello!</p>

var p = document.getElementById('hey');
console.log(p);
console.dir(p);

3. Змінні.

var x; - об’явлення (створення) змінної; від слова variable.

var y = 6; - об’явлення змінної та її ініціалізація - присвоєння їй певного значення.

y = 7; - присвоювання нового значення змінній.

Повертати значення - видати результат, стосується операторів, функцій.

Типи даних:

int - цілі числа зі знаком.

float - дробні числа.

char - символи (1 байт).

string - строкові константи.

true, false - булеві значення.

null, undefined, Infinity, NaN - спеціальні типи даних.

var PORT = '8080'; - константа.

 

Строки потрібно брати у лапки (одинарні чи подвійні):

var a = 'hello';
var b = "world";
// в JS краще використовувати одинарні лапки для строк без апострофів
alert('It\'s my cat!');
alert("It's my cat!");
console.log('<p class="big">');
console.log("<p class=\"big\">");

Не плутайте змінні та строки:

var hey = 'bugaga';
alert('hey');
alert(hey);
alert(bugaga); // помилка

 

typeof - повертає тип змінної.

/* оператор */
var s;
console.log(typeof s); // "undefined"
console.log(typeof 0); // "number"
console.log(typeof 3.5); // "number"
console.log(typeof true); // "boolean"
console.log(typeof 'foo'); // "string"
/* функція */
console.log(typeof(0)); // "number"
console.log(typeof(true)); // "boolean"
console.log(typeof('foo')); // "string"

4. Оператори.

Коментарі:

/* Коментар на дві строки:
document.write("hello!"); */
var i=6; // коментар до кінця строки

Чи потрібно коментувати код, пояснюючи що робить той чи інший блок, та чи інша строка?

Я зустрічав доволі фанатичних розробників, в їхньому коді пояснень більше, ніж самого коду.

В коді варто залишати:

  1. коментарі, які відповідають на питання "чому?", а не "що?", такі коментарі пояснюють незрозумілі ділянки, а не просто описують код;
  2. коментарі перед великими блоками коду, в яких в декількох словах пояснюється що робить даний шмат коду. Це полегшує читання і навігацію по коду;
  3. коментарі в складних структурах даних, які б пояснювали що містить кожне поле в неочевидних випадках;
  4. коментарі, що описують закоментовані шматки коду: чому цей шматок коду закоментований і коли слід безповоротно видалити цей шматок коду.

Оператори:

r = a + b; - додавання.

r = a - b; - віднімання.

r = a * b; - множення.

r = a / b; - ділення (звичайне, дробове).

r = a % b; - залишок від ділення (19 % 4 = 3 тому що 16 / 4 = 4 і 3 у залишку).

i++, ++i - інкремент числа (i = i + 1).

i--, --i - декремент числа (i = i - 1).

Протестуйте наступний код, що показує різницю між останніми операторами та проаналізуйте результати:

console.clear();
console.groupCollapsed('Testing ++ and --');

var a = b = 5;
console.log(a, b);
console.log(a++, ++b, '++');
console.log(a, b);
console.log(a--, --b, '--');
console.log(a, b);

console.groupEnd();

Якщо потрібно збільшити чи зменшити змінну на певне значення, то можна використовувати скорочену форму:

a += 5;    // a = a + 5;
b -= a;    // b = b - a;
c *= b;    // c = c * b;
d /= 2;    // d = d / 2;
e %= 7;    // e = e % 7;

За допомогою оператора + можна об'єднувати строки (конкатинація строк):

var a = 'Hello' + " " + 'world!';
var b = a + '!!';
console.log(a, b, c);

Увага! При додаванні строк і чисел останні перетворюються в строку.

console.log('hello' + 5);
console.log('2+5=' + 2 + 5);
// використовуйте дужки для визначення пріоритетів операцій:
console.log('2+5=' + (2 + 5));

Перетворення типів даних:

console.log(typeof (2 + ''));
var k = 2 + '5';
console.log(k, typeof k);

При інших математичних діях строки намагаються перетворитися на число:

var m = 7 - '5';
var n = '7' * 5;
var p = '8' / '2';
var r = '34a' - 2;
console.log(k, m, n, p, r);

Перетворити строку в число можна функцією parseInt або плюсом:

console.log(k, parseInt(k));
console.log(k, +k);
