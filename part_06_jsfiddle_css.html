1. Робота з jsFiddle.

jsFiddle - пісочниця для веб-девелоперів. Для повноцінної роботи необхідно зареєструватися і залогінитись.

Ця пісочниця являє собою 4 області: для введення блоків коду HTML, CSS, JS і для відображення результату.

Зображення 1. Інтерфейс сервісу jsFiddle.

Після того, як ви ввели потрібні данні - можна отримати досить коротке посилання на цю роботу, щоб поділитися ним з іншими.

jsFiddle використовують для запитань виду "чого в мене не працює, неправильно відображається", щоб показати реалізацію фрагментів верстки, для спільної роботи над проблемою.

Останнім часом з’явилася можливість спільної роботи - Collaboration.

Завдання: зареєструватися і залогінитися на сайті, спробувати вставляти шматки коду зі своїх проектів, перевірити як працює режим Collaboration.

2. Стилі браузера (user agent stylesheet).

По замовчуванню кожен браузер має власний набір стилів для всіх елементів.

Саме тому "голий" HTML виглядає не як текст в блокноті, а має деяке форматування: заголовки виконані більшим і жирним шрифтом, посилання виділені синім кольором, заголовки і параграфи мають відступи, списки також мають відступи та декоративні кружечки на початку кожного елемента списку.

Зображення 2. Стилі браузера.

Проблема в тому, що ці стилі у різних браузерах можуть різнитися.

Перед тим, як почати писати стилі бажано провести обнулення стилів браузера.

Ось приклад обнуляючих стилів, включаючи веб-форми.

html { margin:0; padding:0;}
body { margin:0; padding:0; color:#666; font:normal 18px/28px Helvetice, Arial, sans-serif; background:#fff;}
.clear { clear:both;}
p { margin:0; padding:10px 0;}
a { color:#0000ff; text-decoration:underline;}
a:hover { color:#7777ff;}
a img { border:none;}
h1, h2, h3, h4, h5, h6 { margin:0; padding:0; font-weight:bold; color:#000;}
h1 { padding:40px 0 24px; line-height:1.4em;}
h2 { padding:0 0 12px;}
strong { font-weight:bold;}
em { font-style:italic;}
form, menu, iframe { display:block; margin:0; padding:0; border:none;}
ol, ul, dl, li, dt, dd { margin:0; padding:0; list-style:none;}
* { outline:none;}

3. Стилі шрифта.

Стилі шрифта:

color: #fff;
font-family: "Times New Romain", Helvetica, serif;
font-weight: bold;
font-style: italic;
font-size: 16px;
line-height: 1.4em;

// комплексний стиль:
font: normal 16px/1.4em "Times New Romain", Helvetica, serif;

text-align: center;
text-decoration: underline;

4. Одиниці вимірювання, формат кольору.

Розміри можна задавати в пікселях, типографських точках (поінтах), метричних та імперських одиницях, відсотках.

height: 20px;
width: 50%;
font-size: 16pt;
line-height: 1.3em;
padding: 1cm 0.2inc 0;

Висоту у відсотках для елементів можна задавати тільки якщо вони спозиціоновані абсолютно чи фіксовано.

Після нуля одиниці виміру не ставляться.

Відсотки від розміру вікна браузера (viewport width, viewport height):

width: 80vw;
height: 50vh;

Кутові одиниці виміру:

transform: rotate(360deg); // градуси
transform: rotate(250grad); // 360 градусів = 400grad, прямий кут = 100grad
transform: rotate(3.14rad); // радіани
transform: rotate(0.5turn); // 1 turn = 360 градусів

Колір можна задавати чотирма способами:

color: green;
border-color: #333aaa;
background: rgb(127, 255, 0);
background-color: rgba(255, 0, 160, 0.3);

Використовуйте маленькі літери.

Якщо при заданні кольора в шіснадцятковому вигляді пари цифр/букв повторюються - колір бажано записати у скороченому виді, трьома символами:

color: #33ff99;
color: #3f9;

5. Розміри елемента:

Кожен елемент має ширину та висоту. Блочним елементам можна задавати розміри:

width: 100%;
height: 75px;

Намагайтеся уникати використання height, нехай блоки самі собі виставляють висоту. Використовуйте внутрішні відступи padding та висоту строки line-height для керування висотою блоків.

height: 40px;

6. Відступи:

Відступи бувають зовнішні та внутрішні.

margin: 0 8px 10px 20%;
padding: 10px 8px;

Відступи можуть мати 1, 2, 3, або 4 параметра, що записуються через пробіл:
1 - всі відступи однакові;
2 - вертикальні та горизонтальні відступи;
3 - верхній, горизонтальні, нижній відступи;
4 - верхній, правий, нижній, лівий відступи.

Внутрішні відступи збільшують розміри елемента.

В першу чергу намагайтесь використовувати padding.
