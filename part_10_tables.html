1. Таблиці HTML.

Для виведення табличних даних в HTML існує декілька тегів:

<table> - таблиця;
<tr> - строка таблиці;
<td> - чарунка таблиці;
<th> - чарунка заголовка таблиці;
<thead> - заголовок таблиці;
<tbody> - тіло таблиці. <table>
  <tr>
    <th>Продукція</th>

Атрибути тега <table>:

cellspacing - задає зовнішні відступи довкола чарунок.

cellpadding - задає внутрішні відступи всередині чарунок.

Зазвичай їх обнуляють. Можна зробити гарний бордер у таблиці, якщо їй задати у фон колір бордера, зовнішні відступи встановити на 1-2, чарункам задати потрібний фоновий колір.

8.2. Таблиці CSS.

Стилями можна перетворити будь-які блоки на таблиці. Використовуються такі стилі:

display: table;
display: table-cell;
display: table-row;

Якщо потрібно використати псевдо-табличну верстку для одного рядка - table-row можна опустити.

<p>
  <span>Комірка 1</span>
  <span>Комірка 2</span>
  <span>Комірка 3</span>
</p>

p { display: table;} span { display: table-cell;}
